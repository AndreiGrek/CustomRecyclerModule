# Dependencies
In `build.gradle` module level implementation:
```groovy
implementation: 'com.github.AndreiGrek:CustomRecyclerModule:x.y.z'
```
In `settings.gradle` add `mavenCentral()` repository:
``` groovy
pluginManagement {
    repositories {
        gradlePluginPortal()
        google()
        mavenCentral()
    }
}
```
